# 器具列表变更信息独立列设计

## 目标

将器具列表中目前附加在“证书编号”单元格内的变更状态拆分为独立的“变更信息”列，使证书编号和变更历史入口各自表达单一信息。

## 界面行为

- “证书编号”列只显示证书编号；缺失时显示 `-`。
- 在“证书编号”右侧新增“变更信息”列，宽度约 230px。
- 存在 `latest_change_at` 时，新列显示绿色“已更新”标签、最近变更时间和字段摘要。
- 点击标签或时间摘要打开现有的器具历史抽屉。
- 没有变更历史时，新列显示 `-`，不提供点击操作。

## 实现范围

仅调整 `client/src/views/InstrumentList.vue` 的表格列结构和必要样式，不改变接口、数据库字段、历史抽屉或变更记录逻辑。

- 运行相关前端测试。
- 执行前端生产构建，确认 Vue 模板和样式可正常编译。
