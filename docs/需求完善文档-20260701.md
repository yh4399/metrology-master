## 需求三：证书匹配逻辑修复

### 问题诊断

#### 当前匹配逻辑（有缺陷）

在 [certificate.js](server/routes/certificate.js#L97-L248) 的 `/api/certificate/batch-upload` 中：

1. 从文件名提取信息（第 109、116 行）：
   ```
   originalName = "WB-H1-ZX1-2026062801-4190286"  （去掉 .pdf）
   ```
2. 存储证书编号时（第 116、184 行）：
   ```js
   certificateNumber: originalName  // = "WB-H1-ZX1-2026062801-4190286"
   ```
   直接把整个文件名（去掉 .pdf）作为证书编号存入数据库。

#### 实际的命名规则

以参考文件 `WB-H1-ZX1-2026062801-4190286.pdf` 为例：

| 部分 | 含义 | 示例值 |
|------|------|--------|
| `WB` | 仪表类别前缀 | 温度变送器 |
| `H1-ZX1` | 固定标识 | 中心一号 |
| `2026062801` | **证书编号**（10位日期码+序号） | 2026年06月28日第01号 |
| `4190286` | **出厂编号**（仪表序列号） | 4190286 |

PDF 文档内页实际证书编号也是 `WB-H1-ZX1-2026062801`（不含出厂编号后缀）。

#### 缺陷

当前代码存储的证书编号为 `WB-H1-ZX1-2026062801-4190286`（**包含了出厂编号**），但正确的证书编号应为 `WB-H1-ZX1-2026062801`（**不含出厂编号**）。

这与 PDF 文档内部的「证书编号」字段不一致，也与标准证书命名规范不符。

### 需要修改

在 `server/routes/certificate.js` 的批量上传处理中：

**对于 H1-ZX1 标准格式**（`{PREFIX}-H1-ZX1-{DATECODE}-{SERIAL}`）：

- 证书编号应为：`{PREFIX}-H1-ZX1-{DATECODE}`
  - 即去掉最后的出厂编号段
  - 例：`WB-H1-ZX1-2026062801-4190286` → 证书编号 = `WB-H1-ZX1-2026062801`

- 出厂编号应为：`{SERIAL}`（最后一段）
  - 例：`4190286`（当前逻辑对此已正确处理）

**关键修改点**：

1. **第 109-116 行附近**：新增提取「正确证书编号」的逻辑
   - 对于 H1-ZX1 格式：证书编号 = `{PREFIX}-H1-ZX1-{DATECODE}`
   - 对于非标准格式：证书编号 = 文件名（当前逻辑，保持不变）
   
2. **第 184、212 行**：`{ certificate_number: originalName }` 改为使用正确提取的证书编号

### 涉及文件

- `server/routes/certificate.js` — 第 109-116 行（文件名解析）、第 184 行（单条匹配写入）、第 212 行（多条匹配写入）

### 当前导出逻辑（需改造）

| 导出端点 | 当前格式 | 问题 |
|---------|---------|------|
| `/api/instruments/export` | 通用 17 列固定表头，单 Sheet | **不按类别分 Sheet，列顺序与模板不一致** |

| `/api/instruments/export/warning-apply` | 检定申请表格式 | 单独保留，不在本次修改范围 |

### 需要修改

#### 核心原则

1. **按类别自动分组** — 导出的仪表根据 `category` 字段分到对应的 Sheet
2. **每个 Sheet 匹配模板列** — 列标题、列顺序必须与模板一致
3. **标题行格式一致** — 第一行为合并的台账标题（如"海一采油管理区压力变送器台帐"）
4. **序号列使用公式** — `=ROW()-2`（与模板一致）
5. **日期格式** — 统一为 `YYYY.MM.DD` 格式
6. **空类别/未匹配的 Sheet** — 模板中存在但数据库无数据的 Sheet 也保留（空表）

#### 数据库字段 → 模板列的映射关系（核心类别）

**压变/温变** 通用映射：

| DB 字段 | 模板列 |
|---------|-------|
| `serial_number` | 出厂编号 / 序列号 |
| `installation_location` | 安装位置 |
| `model` | 规格型号 |
| `manufacturer` | 生产厂家 |
| `range_info` (或 range_min/max/unit 拼接) | 量程（MPa/℃） |
| `mfg_date` | 出厂日期 |
| `inspection_date` | 检验时间 / 检定时间 |
| `valid_until` | 有效期 |
| `certificate_number` | 仪表检定证书编号 |
| `classification` | 分类 |
| `inspection_unit` | 检验单位 |
